# Prpr::BranchWarning

[Prpr](https://github.com/mzp/prpr) plugin to add a label to pull requests that merge to the not default branch

## Install

```ruby
gem 'prpr-branch_warning', github: 'misoca/prpr-branch_warning'
```

## ENV

 * `BRANCH_WARNING_LABEL`: label title
